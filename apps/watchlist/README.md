# Watchlist Tab Panel (ISW-11)

富途牛牛风格的自选股分类 Tab 面板，作为投研一体机四象限布局的右上面板。

## 运行

无构建步骤，纯静态资源。任选其一：

```bash
# 从仓库根目录起一个本地 HTTP 服务
python3 -m http.server 8080
# 然后访问 http://localhost:8080/apps/watchlist/
```

或直接用浏览器打开 `apps/watchlist/index.html`。

## 功能

- 顶部 Tab 分类：科技 / 能源 / 金融 / ETF，支持「新增 / 重命名 / 删除」（点击 Tab 右侧的 `⋯`）。
- 拖拽股票行到其他 Tab 即可重新归类。
- 每行显示：股票名称 + 代码 | 最新价 | 涨跌额 | 涨跌幅。涨跌幅用红/绿色块标识。
- 配色切换：「A股 红涨」 / 「美股 绿涨」。
- 点击股票行从右侧滑出详情面板：最新价、涨跌、昨收/今开/最高/最低、SVG 迷你走势图。
- 开盘时间内（工作日 9:30–16:00 本地时间）每 30 秒自动刷新报价；右上角「刷新」按钮可手动触发。
- 所有状态（Tab、归类、配色、最近报价）存入 `localStorage`，刷新页面后保留。

## 验收对应

- ≥ 3 个 Tab 分类：默认 4 个，且支持新增。
- 实时显示价格与涨跌：模拟报价每 30 秒漂移，并即时重绘。
- 红绿色块区分涨跌：见 `stock-row__pct.is-up / .is-down`，红涨绿跌为默认配色。
- 点击股票看到详细行情：见 `#detail-panel`。

## 数据说明

报价为本地模拟（漂移 ±0.5%），未接入真实行情。生产化时可将 `tickQuotes()` 替换为外部行情源调用，
并把 `state.lastUpdated` 与 UI 解耦走 WebSocket/SSE。
